Sobre Nosotros

El sistema utiliza la arquitectura REST (Firebase) para gestionar inventario, tareas y métricas poblacionales en tiempo real.

Integración API REST

Toda la gestión de datos se realiza mediante peticion gi **`fetch()`** asíncronas a la base de datos de Firebase Realtime Database.

Diseño Responsivo

Uso de Flexbox y Media Queries en **`diseño.css`** para garantizar la correcta visualización en dispositivos móviles y de escritorio.

Estructura Semántica

Implementación correcta de etiquetas HTML5 estructurales.
